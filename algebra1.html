Unit 1: Algebra Fundamentals

Introduction to variables

Variables are letters or symbols that stand for numbers we don't yet know or haven't specified yet. They are fundamental tools in algebra that allow us to describe patterns, write general formulas, and solve problems without always requiring specific numerical values. Variables serve multiple important roles:

  • Unknowns to solve for: Used to represent missing values we need to find (e.g., x in 2x = 10)
  • Parameters that vary: Stand-in values that can change and affect the output of an expression (e.g., temperature t in a formula)
  • General representation: Allow us to express patterns and relationships that apply to many different situations
Example 1: Let x be the number of apples. If x = 5, then x + 2 = 7.
Example 2: Let t represent temperature in °C. The expression 2t + 1 gives a value that depends on t: if t = 10, 2t+1 = 21.

Evaluating expressions with one variable

To evaluate an expression with one variable, substitute the variable with the given number and then compute the result carefully using the order of operations. This systematic approach ensures accurate calculations regardless of the complexity of the expression. Key steps include:

  • Replace the variable with the given numerical value, using parentheses if needed
  • Follow the correct order of operations: PEMDAS (Parentheses, Exponents, Multiplication/Division, Addition/Subtraction)
  • Pay special attention to negative numbers and ensure parentheses are handled correctly to avoid sign errors
Example 1: Evaluate 3x + 2 when x = 5: 3(5) + 2 = 15 + 2 = 17.
Example 2: Evaluate 3(x - 2) + 4 when x = 5: 3(3) + 4 = 9 + 4 = 13.

Evaluating expressions with two variables

When expressions have two variables, you must substitute the correct value for each distinct variable before evaluating. The process is similar to single-variable evaluation but requires careful tracking of which value corresponds to which variable. Pay special attention to:

  • Implicit multiplication: When variables are next to each other (like xy), they are being multiplied together
  • Variables as exponents: If a variable appears as an exponent (like x^y), calculate the exponent first, then apply it
  • Order of operations: Still apply PEMDAS after substituting both variables
Example 2: If x = 4 and y = 1, x + 2y = 4 + 2(1) = 6.

Evaluating expressions with two variables, fractions and decimals

When substituting fractions and decimals into algebraic expressions, all previous substitution and evaluation rules apply without change. You have flexibility in your approach and can choose the method that feels most natural:

  • Fraction method: Add, subtract, multiply, and divide fractions directly by finding common denominators when needed
  • Decimal method: Convert all fractions to decimals and perform calculations in decimal form
  • Mixed method: Convert only when necessary for simplification or clarity
  • Rounding awareness: Keep track of when results are approximations versus exact values
Example 1: If x = 1/2 and y = 0.5, x + y = 0.5 + 0.5 = 1.
Example 2: If x = 1/4 and y = 0.5, x + y = 0.25 + 0.5 = 0.75.

Unification of like terms

Like terms are terms in an algebraic expression that have identical variable parts. This means they must contain the exact same variables raised to the exact same powers. Combining like terms is a fundamental skill that simplifies expressions and is essential for solving equations. To combine like terms:

  • Identify like terms: Look for terms with the same variable(s) and exponent(s) (e.g., 3x and 5x are like; 3x and 3x² are not)
  • Group them together: Rearrange the expression to place like terms adjacent to each other
  • Add/subtract coefficients: Combine by adding or subtracting only the numerical coefficients while keeping the variable part unchanged
  • Constants as like terms: Numbers without variables are all like terms with each other
Example 1: 3x + 5x = (3 + 5)x = 8x.
Example 2: 2x^2 + 3x^2 = 5x^2. Note that 3x and 3x^2 are not like terms because the powers differ.

Equivalent expressions

Equivalent expressions are two or more expressions that represent the same mathematical value for all possible values of the variables. Understanding equivalence is crucial for solving equations and simplifying complex expressions. You can demonstrate equivalence through two main approaches:

  • Algebraic manipulation: Transform one expression into another using valid operations such as the distributive property, factoring, combining like terms, or expanding products
  • Numerical verification: Test both expressions using several different variable values and verify that they produce identical results in each case
  • Why it matters: Different forms of equivalent expressions often reveal different insights about a problem's structure
Example 1: 2(x + 3) and 2x + 6 are equivalent because distributing gives 2x + 6.
Example 2: (x + 1)^2 and x^2 + 2x + 1 are equivalent after expanding the square.

Division by zero / undefined or undetermined expressions

Division by zero is undefined and cannot be evaluated in standard arithmetic. When we encounter expressions containing variables in denominators, we must identify which input values would make the denominator zero and exclude those from the domain. This concept is essential for understanding the domain of rational expressions. Key distinctions include:

  • Simple undefined cases: Any expression like 5/0 or 1/(x-2) when x=2 has no meaningful value
  • Indeterminate form (0/0): Expressions like (x²-4)/(x-2) produce 0/0 when x=2, suggesting a removable discontinuity or common factor that needs investigation
  • Domain restrictions: Always identify which input values make any denominator equal zero and exclude those values from the domain
  • Removable vs. non-removable: 0/0 forms may simplify after factoring, while other undefined cases represent permanent gaps in the function
Example 1: The expression 1/(x - 2) is undefined when x = 2 because the denominator is zero.
Example 2: (x^2 - 4)/(x - 2) gives 0/0 when x = 2, but after factoring (x-2)(x+2)/(x-2) it simplifies to x+2 for x ≠ 2; the original expression is still undefined at x = 2 (a "hole").

3) For (x^2-4)/(x-2), is x = 2 allowed? (answer: no)

Unit 2: Solving Equations and Inequalities

Linear equations with variables on both sides

When solving equations with variables on both sides, the fundamental principle is keeping the equation balanced—whatever operation you perform must be applied to both sides equally. The strategy is to systematically isolate the variable by collecting all variable terms on one side and constant terms on the other side. Here's the systematic approach:

  • Collect variable terms: Move all terms containing the variable to one side using inverse operations (typically subtraction if adding, or addition if subtracting)
  • Collect constants: Move all number-only terms to the opposite side using the same inverse operation principle
  • Isolate the variable: Divide both sides by the coefficient to get the variable alone
  • Verify your solution: Substitute back into the original equation to confirm both sides equal the same value
Example 1: Solve 3x + 5 = 2x + 9. Subtract 2x from both sides: x + 5 = 9. Subtract 5: x = 4.
Example 2: Solve 4x - 2 = 2x + 6. Subtract 2x: 2x - 2 = 6. Add 2: 2x = 8. Divide by 2: x = 4.

Linear equations with parenthesis

Equations containing parentheses require you to expand them fully before collecting like terms and solving. The distributive property is the key tool here, which states that a(b + c) = ab + ac. Understanding the order of operations when parentheses are involved is crucial for accurate solutions. Follow these steps:

  • Distribute first: Multiply the coefficient outside the parentheses by each term inside using the distributive property
  • Simplify each side: Combine like terms on both sides of the equation
  • Move variable terms: Collect all variables on one side and constants on the other
  • Solve for the variable: Isolate the variable by dividing by its coefficient
Example 1: Solve 2(x + 3) = 10. Distribute: 2x + 6 = 10. Subtract 6: 2x = 4. Divide by 2: x = 2.
Example 2: Solve 3(x - 1) + 2 = 11. Distribute: 3x - 3 + 2 = 11. Simplify: 3x - 1 = 11. Add 1: 3x = 12. Divide by 3: x = 4.

Number of solutions

Linear equations are remarkable because they can have exactly one solution, no solutions, or infinitely many solutions. Understanding which case you're in requires paying attention to what happens when you simplify the equation. Here's how to identify each type:

  • One unique solution: Most linear equations have one answer; you'll get a variable equal to a number (e.g., x = 5)
  • Infinitely many solutions: When simplifying leads to a true statement (like 5 = 5 or 0 = 0), every value of the variable is a solution—the two equations represent the same line
  • No solution: When simplifying leads to a false statement (like 5 = 3 or 0 = 7), there's no value of the variable that makes the equation true—the equations represent parallel but distinct lines
Example 1: Solve x + 3 = x + 3. Subtract x: 3 = 3. This is true, so infinitely many solutions (the equation is always true).
Example 2: Solve x + 2 = x + 5. Subtract x: 2 = 5. This is false, so no solution (the equation is never true).
Practice (challenge):
1) Solve: 2x = 2x. How many solutions? (answer: infinitely many)
2) Solve: x + 1 = x + 2. How many solutions? (answer: none)
3) Solve: 3x = 12. How many solutions? (answer: one)

Unknown coefficients

Sometimes you encounter equations where a coefficient is unknown and you need to find its value given additional information about the variable. This type of problem reverses the typical solving process; instead of finding the variable, you're finding the coefficient. The solving method remains the same:

  • Substitute the known variable: Replace the variable with its given value
  • Simplify: Perform the mental arithmetic with the known value
  • Solve for the coefficient: Use inverse operations to isolate the unknown coefficient just as you would isolate a variable
  • Verify: Substitute both values back into the original equation to confirm correctness
Example 1: If ax + 2 = 8 and x = 2, find a. Substitute: a(2) + 2 = 8. Simplify: 2a + 2 = 8. Subtract 2: 2a = 6. Divide: a = 3.
Example 2: If bx - 1 = 5 and x = 3, find b. Substitute: b(3) - 1 = 5. Add 1: 3b = 6. Divide: b = 2.

Inequalities with various steps

Inequalities are solved almost identically to equations, with one critical exception that students must remember: the inequality symbol flips when you multiply or divide by a negative number. This rule exists because multiplying by a negative reverses the order of numbers on the number line. The complete process involves:

  • Use inverse operations: Add, subtract, multiply, and divide exactly as in equation-solving to isolate the variable
  • Flip the symbol: Whenever you multiply or divide both sides by a negative number, reverse the inequality symbol (< becomes >, ≤ becomes ≥, and vice versa)
  • Express the solution: Write the final answer in inequality form (e.g., x > 3) or in interval notation
  • Verify with test points: Choose values on both sides of your boundary and verify they satisfy or don't satisfy the inequality as expected
Example 1: Solve 2x + 3 < 9. Subtract 3: 2x < 6. Divide by 2: x < 3.
Example 2: Solve -x + 5 ≥ 2. Subtract 5: -x ≥ -3. Multiply by -1 (flip): x ≤ 3.

Compound inequalities

Compound inequalities combine two separate inequalities using either "and" or "or". These connectors create fundamentally different solution sets. Learning to work with both types is essential for describing ranges and excluding regions on the number line. The key differences are:

  • "And" inequalities (conjunctions): Both conditions must be true simultaneously (e.g., 2 < x < 5 means x is between 2 and 5 inclusive); solve by treating all three parts equally with the same operations
  • "Or" inequalities (disjunctions): Either condition can be true, representing two separate regions on the number line (e.g., x < 2 or x ≥ 5 means x is less than 2 or at least 5); solve each inequality separately
  • Graphing compound inequalities: For "and", shade the overlapping region; for "or", shade both separate regions
Example 1: Solve 2 < x + 1 < 5. Subtract 1 from all parts: 1 < x < 4.
Example 2: Solve x < 2 or x ≥ 5. This represents two separate regions: all values less than 2, and all values 5 or greater.

Unit 3: Working with Units

Conversion rates

A conversion rate (or conversion factor) is a ratio that expresses the relationship between two different measurement units. The key to successful unit conversion is dimensional analysis: arranging the conversion ratio so that the original unit cancels out algebraically, leaving only the desired unit. Understanding conversion factors allows you to:

Example 1: Convert 5 kilometers to meters. Use the conversion rate: 1 km = 1000 m. So 5 km × (1000 m / 1 km) = 5000 m.
Example 2: Convert 60 miles per hour to feet per second. Use: 60 mi/hr × (5280 ft/mi) × (1 hr/3600 s) = 88 ft/s.

Appropriate units

Choosing the right measurement unit is crucial for clear communication and practical understanding of quantities. Different situations call for different scales. A well-chosen unit makes the resulting number reasonable and easy to interpret. Guidelines for selecting appropriate units include:

  • Avoid extreme values: Choose units that produce numbers in a reasonable range (heights in meters, not kilometers; pencil width in millimeters, not meters)
  • Match the context: Scientific precision might require smaller units; everyday conversation often uses larger, more familiar units
  • Compare similar quantities: When comparing measurements, use the same unit for fairness and clarity
  • Consider precision: The unit choice should match the accuracy of your measurement
Example 1: To measure the height of a person, use meters or centimeters, not kilometers. A height of 1.75 m is more sensible than 0.00175 km.
Example 2: To measure the thickness of paper, use millimeters, not meters. Paper thickness is about 0.1 mm, which is clearer than 0.0001 m.

Verbal problems with varied units

Real-world word problems often involve quantities expressed in different units, requiring you to convert before combining or comparing them. The systematic approach ensures accuracy and helps you catch errors. When solving problems with multiple units:

  • Identify all quantities and their units: Write them down clearly to avoid confusion
  • Choose a target unit: Decide what unit the final answer should be in
  • Convert all quantities to that unit: Use appropriate conversion factors, applying dimensional analysis carefully
  • Set up and solve the equation: Perform calculations as usual
  • Include units throughout: Write units with every number to track what you're calculating and verify the answer makes sense
  • Check reasonableness: Does your answer's magnitude make sense? Are the units correct?
Example 1: A recipe calls for 250 mL of water and 2 liters of milk. How much liquid total in liters? Convert 250 mL to liters: 250 mL ÷ 1000 = 0.25 L. Total: 0.25 L + 2 L = 2.25 L.
Example 2: A car travels 60 km in 1.5 hours. What is the speed in km/h? Speed = Distance ÷ Time = 60 km ÷ 1.5 h = 40 km/h.

Unit 4: Linear Equations and Graphs

Linear equations with two variables

A linear equation with two variables establishes a relationship between two quantities, and when graphed, always produces a straight line. Understanding this relationship is essential for analyzing dependencies between variables. These equations can be expressed in multiple equivalent forms, each revealing different information:

  • Slope-intercept form (y = mx + b): Shows the slope and y-intercept directly; easiest for graphing and understanding growth rate
  • Standard form (Ax + By = C): Uses integers and is useful for finding intercepts algebraically
  • Point-slope form (y - y₁ = m(x - x₁)): Uses a known point and slope; useful when those are what you're given
  • Solutions as points: Any point (x, y) that makes the equation true represents a solution and lies on the corresponding line
Example 1: The equation y = 2x + 1 is linear. The point (0, 1) is on the line because y = 2(0) + 1 = 1. The point (2, 5) is on the line because y = 2(2) + 1 = 5.
Example 2: The equation 3x + 2y = 12 is linear in standard form. The point (2, 3) is on the line because 3(2) + 2(3) = 6 + 6 = 12.
Practice (challenge):
1) For y = 3x - 2, is the point (1, 1) on the line? (answer: yes)
2) For y = -x + 4, what is y when x = 2?
3) For 2x + y = 8, is the point (3, 2) on the line? (answer: yes)

The Slope

Slope is a fundamental measure of how a line tilts, representing the rate of change between any two points. It's calculated as the ratio of vertical change (rise) to horizontal change (run): m = (y₂ - y₁) / (x₂ - x₁). Understanding slope allows you to predict outputs, compare rates, and interpret real-world change. Key slope concepts include:

  • Positive slope: The line rises from left to right, indicating an increasing relationship
  • Negative slope: The line falls from left to right, indicating a decreasing relationship
  • Zero slope: A horizontal line where y stays constant regardless of x
  • Undefined slope: A vertical line where x is constant; division by zero occurs because there's no horizontal change
  • Interpretation: The slope value tells you how much y changes for each unit increase in x
Example 1: Find the slope between (1, 2) and (3, 6). m = (6 - 2) / (3 - 1) = 4 / 2 = 2. The slope is 2, meaning for every 1 unit right, y increases by 2.
Example 2: Find the slope between (2, 5) and (4, 1). m = (1 - 5) / (4 - 2) = -4 / 2 = -2. The slope is -2, a negative slope.

Horizontal and vertical lines

Horizontal and vertical lines are special cases that don't follow the typical slope formula. Understanding these lines is important because they have unique properties that affect equation-solving and graphing. Here's what distinguishes them:

  • Horizontal lines: Expressed as y = c (where c is a constant), they have a slope of 0 because there is zero vertical change; every point has the same y-coordinate
  • Vertical lines: Expressed as x = c, they have undefined slope because calculating rise/run requires division by zero; every point has the same x-coordinate
  • Function test: Vertical lines fail the vertical line test and are not functions (one x-value maps to multiple y-values)
  • Graphing: Horizontal lines are easy to graph; vertical lines require special handling since they can't be written in slope-intercept form
Example 1: The equation y = 5 represents a horizontal line. Every point has y-coordinate 5, like (0, 5), (2, 5), (-3, 5). The slope is 0.
Example 2: The equation x = -2 represents a vertical line. Every point has x-coordinate -2, like (-2, 0), (-2, 3), (-2, -1). The slope is undefined.

2) What is the form of a vertical line? (answer: x = c)

Intersections in x-axis / y-axis

Intercepts are the points where a line crosses the coordinate axes. These points are particularly important because they often have real-world meaning and provide useful information for graphing. Finding intercepts requires a systematic approach:

  • X-intercept (zero of function): The point where the line crosses the x-axis; occurs when y = 0. Set y = 0 in the equation and solve for x. Written as (x,0)
  • Y-intercept: The point where the line crosses the y-axis; occurs when x = 0. Set x = 0 in the equation and solve for y. Written as (0,y)
  • Graphing utility: These two points alone are usually sufficient to draw an accurate line graph
  • Relationship to forms: In y = mx + b form, b is the y-intercept; in standard form Ax + By = C, intercepts come from setting x and y to zero
Example 1: For y = 2x + 4, find the y-intercept: set x = 0, y = 2(0) + 4 = 4. y-intercept is (0, 4). Find x-intercept: set y = 0, 0 = 2x + 4, so x = -2. x-intercept is (-2, 0).
Example 2: For 3x - y = 6, find y-intercept: set x = 0, -y = 6, so y = -6. y-intercept is (0, -6). Find x-intercept: set y = 0, 3x = 6, so x = 2. x-intercept is (2, 0).
Practice (challenge):

Application of intersections and slope

Linear relationships appear throughout the real world, and understanding how to interpret slope and intercepts in context allows you to extract meaning from mathematical models. The parameters of a linear equation often represent real quantities with practical significance:

  • Y-intercept meaning: Often represents an initial value, starting amount, or fixed cost before any variable activity occurs
  • Slope meaning: Represents a rate of change — how much one quantity changes per unit change in another (cost per item, distance per hour, growth per year)
  • X-intercept meaning: Often represents when something reaches zero, becomes unprofitable, or concludes (when debt is paid off, when a tank empties, break-even point)
  • Building context equations: From a real situation, identify these values and write the appropriate linear equation
  • Predicting values: Use the equation to answer "what-if" questions about values beyond your original data
Example 1: A phone plan costs $20 monthly plus $0.10 per minute. The equation is y = 0.10x + 20. The y-intercept (20) is the base monthly cost. The slope (0.10) is the cost per minute. If x = 100 minutes, the cost is y = 0.10(100) + 20 = $30.
Example 2: A car depreciates at $2000 per year, starting at $25000. The equation is y = -2000x + 25000. The y-intercept (25000) is the initial price. The slope (-2000) is the yearly depreciation. The x-intercept (when y = 0) is when the car is worth nothing: 0 = -2000x + 25000, so x = 12.5 years.

Unit 5: Forms of linear equations

Slope-Intercept form

Slope-intercept form is the most popular form for writing linear equations because it clearly displays both essential pieces of information about the line. The standard format is y = mx + b, where each parameter has a specific meaning. This form's clarity makes it ideal for graphing, comparing lines, and understanding behavior:

  • m is the slope: The rate of change; how much y increases (or decreases if negative) for each unit increase in x
  • b is the y-intercept: The y-coordinate where the line crosses the y-axis; occurs when x = 0
  • Graphing advantage: Plot the point (0, b), then use the slope to find more points by moving right and up (or down if slope is negative)
  • Easy to identify: Comparison between equations is straightforward—lines with the same m are parallel; lines with different m intersect
Example: y = 2x + 1 has slope m = 2 and y-intercept b = 1.

Graphing slope-intercept equations

Graphing a linear equation in slope-intercept form is systematic and efficient. The format y = mx + b gives you everything needed to quickly create an accurate graph. This skill bridges the gap between algebraic and visual understanding of linear relationships. The process involves:

  • Plot the y-intercept: Mark the point (0, b) on the y-axis
  • Use the slope as a tool: From (0, b), move using the slope rise/run; if m = 2/1, go right 1 and up 2 to get another point
  • Connect the points: Draw a straight line through the points extending across the entire graph
  • Check negative slopes: If m is negative, moving right means moving down; if m is a fraction, the denominator tells you horizontal distance
  • Verify with a second point: Calculate another point using a different x-value to ensure your line is correct
Example: Graph y = 2x + 1 by plotting (0,1) then moving right 1 and up 2 to (1,3).

Writing slope-intercept equations

Writing equations in slope-intercept form requires identifying two key pieces of information: the slope and the y-intercept. The scenarios you'll encounter vary, but the procedure remains consistent. Understanding how to extract parameters from different given information is essential:

  • Given slope and y-intercept: Directly substitute m and b into y = mx + b
  • Given slope and a point: Substitute the slope and point into y = mx + b, solve for b, then write the complete equation
  • Given two points: Calculate slope using m = (y₂ - y₁)/(x₂ - x₁), then use one point to find b, and write the equation

Point-slope form

Point-slope form is y - y₁ = m(x - x₁), where (x₁, y₁) is a known point on the line and m is the slope. This form is particularly powerful when you have a specific point and the slope, as you can write the equation without first finding the y-intercept. Understanding this form provides flexibility in equation-writing:

  • Direct use of given information: If you know a point and slope, use them directly without extra calculation
  • Conversion to slope-intercept: Expand and simplify to get y = mx + b when that form is desired
  • Conceptual clarity: The form emphasizes that the slope between any point (x, y) and the known point (x₁, y₁) must equal m
  • Real-world modeling: Useful when you have an initial point and a known rate of change
Example: With point (2,3) and m=4: y - 3 = 4(x - 2) → y = 4x - 5.

Standard form

Standard form is Ax + By = C, where A, B, and C are integers. This form is algebraically elegant and useful for certain applications, particularly when finding intercepts or working with systems of equations. Each form serves different purposes, and knowing all three gives you flexibility:

  • Intercepts are easy: Set x = 0 for the y-intercept, set y = 0 for the x-intercept; no fraction manipulation needed
  • Standard calculations: Works well with elimination method when solving systems of equations
  • Integer coefficients: Using only integers eliminates fraction concerns in some contexts
  • Conversion between forms: From slope-intercept y = mx + b, rearrange to -mx + y = b, or multiply through to clear fractions and get standard form
  • Graphing note: Standard form is less intuitive for graphing than slope-intercept, so conversion is often necessary

Unit 6: Systems of Equations

Introduction

A system of equations is a set of two or more equations with the same variables. A solution to a system is an ordered pair (or tuple) that satisfies all equations simultaneously. For example, the system y = 2x + 1 and y = -x + 4 has a solution at the point where both lines intersect.

Example: Check if (1, 3) is a solution to y = 2x + 1 and y = -x + 4: First equation: 3 = 2(1) + 1 = 3 ✓. Second: 3 = -(1) + 4 = 3 ✓. Yes, (1,3) is the solution.

Solving with substitution

Substitution method: Solve one equation for one variable, then substitute that expression into the other equation. This reduces the system to a single equation in one variable, which you can solve.

Example: Solve: y = 2x and 3x + y = 10. Substitute: 3x + (2x) = 10 → 5x = 10 → x = 2. Then y = 2(2) = 4. Solution: (2, 4).

Elimination method: Multiply one or both equations so that a variable has opposite coefficients in each equation. Then add the equations to eliminate that variable and solve for the remaining variable.

Equivalent systems

Two systems are equivalent if they have the same solution set. Performing operations like multiplying an equation by a non-zero constant or adding a multiple of one equation to another creates an equivalent system. These operations preserve solutions.

Example: x + y = 3 and 2x + 2y = 6 are equivalent (multiply first by 2). Both have infinite solutions since the second is just a multiple of the first.
Practice:

Number of solutions to a system

A system of two linear equations can have: (1) exactly one solution (lines intersect at one point), (2) infinitely many solutions (lines are the same), or (3) no solution (lines are parallel but distinct).

Example 1: y = 2x + 1 and y = 2x + 3: parallel lines (same slope, different intercepts) → no solution.
Example 2: y = 2x + 1 and 2y = 4x + 2: the second simplifies to y = 2x + 1, so they're the same line → infinitely many solutions.

Verbal problems

Real-world situations often involve systems of equations. Identify the variables, write equations based on the conditions, and solve the system to answer the question.

Example: Alice and Bob have 20 dollars combined. Alice has 4 more than Bob. Let a = Alice's money, b = Bob's money: a + b = 20 and a = b + 4. Substitute: (b+4) + b = 20 → 2b = 16 → b = 8, a = 12. Alice has $12, Bob has $8.

Checking solutions of two-variable inequalities

For a two-variable inequality like y > 2x + 1, check if a point (x, y) is a solution by substituting x and y into the inequality. If the resulting statement is true, the point is a solution.

Example: Is (2, 6) a solution to y > 2x + 1? Substitute: 6 > 2(2) + 1 = 5 → 6 > 5 is true. Yes, (2,6) is a solution.

Domain and range

The domain of a function is the set of all possible input values (x). The range is the set of all possible output values (y). For example, f(x) = √x has domain x ≥ 0 (non-negative reals) and range y ≥ 0.

Example: f(x) = 1/x has domain x ≠ 0 (all reals except 0) and range y ≠ 0.
Practice:
1) For f(x) = √(x - 1), what is the domain? (answer: x ≥ 1)

Finding the domain

To find domain, identify restrictions: (1) Denominators ≠ 0, (2) Expressions under square roots ≥ 0, (3) Logarithm arguments > 0. The domain is all real numbers excluding these restricted values.

Example 1: f(x) = 1/(x - 3). Denominator ≠ 0: x ≠ 3. Domain: all reals except 3, or (-∞, 3) ∪ (3, ∞).
Example 2: f(x) = √(2x - 4). Radicand ≥ 0: 2x - 4 ≥ 0 → x ≥ 2. Domain: [2, ∞).

Recognizing functions

A relation is a function if each input has exactly one output. The vertical line test: if every vertical line intersects the graph at most once, it's a function. If any vertical line intersects twice or more, it's not.

Example: y = x² is a function (vertical line test passes). x = y² is not a function (a vertical line can intersect it twice).

Maximum and minimum of a graph

A maximum is the highest point on a graph (locally or globally). A minimum is the lowest point. For a parabola f(x) = ax², if a > 0 the vertex is a minimum; if a < 0 the vertex is a maximum. The vertex is at x = -b/(2a).

Example: f(x) = x² - 4x + 3. Here a=1 > 0, so minimum at x = -(-4)/(2·1) = 2. f(2) = 4 - 8 + 3 = -1. Minimum: (2, -1).

Increasing, decreasing, positive, and negative intervals

A function is increasing where the graph goes up as x increases, and decreasing where it goes down. It's positive where y > 0 (above x-axis) and negative where y < 0 (below x-axis). Use interval notation like [a,b] or (a,b).

Example: For f(x) = x² - 4: positive on (-∞, -2) ∪ (2, ∞), negative on (-2, 2), decreasing on (-∞, 0), increasing on (0, ∞).

Average rate of change

The average rate of change of f(x) from x = a to x = b is [f(b) - f(a)] / (b - a). It represents the slope of the secant line connecting two points on the graph and shows how fast the function changes on average over an interval.

Example: f(x) = x². From x=1 to x=3: [f(3) - f(1)] / (3 - 1) = [9 - 1] / 2 = 4. Average rate of change is 4.

An arithmetic sequence is a list of numbers where each term is found by adding the same value (the common difference) to the previous term. The constant difference makes the sequence grow or shrink at a steady rate.

Unit 12: Exponential growth and decay

Linear vs exponential growth

Linear growth adds the same amount each time. Exponential growth multiplies by the same factor each time. Exponential change becomes much larger over long periods.

Example: Linear: 2, 4, 6, 8 (add 2). Exponential: 2, 4, 8, 16 (multiply by 2).
Practice:
1) Which is exponential: 3, 6, 12, 24 or 5, 10, 15, 20?

Exponential expressions

Exponential expressions have the form a*b^x. The variable is in the exponent, which makes the rate of change proportional to the current value.

Example: 5*2^x is exponential; 2x + 5 is linear.
Practice:

Growth and decay

Exponential growth: y = a(1 + r)^t. Exponential decay: y = a(1 - r)^t. r is the rate, t is time. The base (1 ± r) is called the growth or decay factor.

Example: A population doubles every year: y = 100*2^t.

Unit 15: Irrational numbers

Introduction

Irrational numbers cannot be written as a ratio of integers. Their decimal expansions never repeat or terminate. They often arise from roots and geometric measurements.

Example: π and √2 are irrational numbers.

Sums and products of rational and irrational numbers

The sum or product of a rational and an irrational number is always irrational (unless the rational is zero for products). This is a key closure property to remember.

Example: 2 + √3 is irrational; 0 × √5 = 0 (rational).
Practice:

Proofs concerning irrational numbers

To prove a number is irrational, show it cannot be written as a fraction. For example, √2 is irrational because no fraction squared equals 2. Many proofs use contradiction and parity arguments.

Example: Proof by contradiction: Assume √2 = a/b in lowest terms. Then 2b^2 = a^2, so a is even, a = 2k. Substitute: 2b^2 = 4k^2 → b^2 = 2k^2, so b is even. Both a and b are even, contradicting lowest terms.
