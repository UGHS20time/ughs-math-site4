Unit 1: Limits and continuity

Intro to limits

A limit describes the value a function approaches as the input approaches a specific value. The key insight is that we care about where the function is heading, not necessarily where it is at that exact point. In notation, lim(x→a) f(x) = L means the function approaches the value L as x gets closer to a. The limit can exist even if f(a) is undefined or different from L.

Example: lim(x→2) (x² − 4)/(x − 2) = 4, even though the function is undefined at x = 2.
Practice:
1) Does lim(x→1) f(x) require f(1) to be defined? (answer: no)

Estimating limits from graphs/tables

Graphically, to find lim(x→a) f(x), look at the y-value the curve approaches as x approaches a from both sides. From a table, examine values to the left and right of a. The limit exists and equals L only if both the left-hand limit and right-hand limit equal L.

Example: If f(1.9)=3.8 and f(2.1)=4.2, the limit near 2 is approximately 4.
Practice:
1) If approaching from the left gives 3 and from the right gives 5, does the limit exist? (answer: no)

Formal definition of limits

The ε-δ definition states: lim(x→a) f(x) = L if for every ε > 0, there exists δ > 0 such that whenever 0 < |x − a| < δ, we have |f(x) − L| < ε. This formalizes "getting arbitrarily close to L" by saying for any small distance ε, we can always find an input window δ around a where outputs stay within ε of L.

Example: For f(x) = 2x, lim(x→1) f(x) = 2 because we can make |2x − 2| small by making |x − 1| small.

Properties of limits

If lim(x→a) f(x) = L and lim(x→a) g(x) = M, then limits respect algebraic operations:

  • Sum: lim(x→a) [f(x) + g(x)] = L + M
  • Difference: lim(x→a) [f(x) − g(x)] = L − M
  • Product: lim(x→a) [f(x) · g(x)] = L · M
  • Quotient: lim(x→a) [f(x)/g(x)] = L/M (provided M ≠ 0)

These properties let you break complex limits into simpler ones.

Example: lim(x→1) (x² + 3x) = lim(x→1) x² + lim(x→1) 3x = 1 + 3 = 4.

Squeeze theorem

If f(x) ≤ g(x) ≤ h(x) for all x near a, and lim(x→a) f(x) = lim(x→a) h(x) = L, then lim(x→a) g(x) = L. This theorem "squeezes" a hard-to-analyze function between two easier functions that converge to the same limit.

Example: lim(x→0) x² sin(1/x) = 0 because −x² ≤ x² sin(1/x) ≤ x² and both bounds approach 0.

Types of discontinuities

  • Removable: A factor cancels or function is undefined, but the limit exists. Appears as a "hole" on the graph.
  • Jump: Left-hand and right-hand limits exist but differ. The function "jumps" at that point.
  • Infinite: Occurs at vertical asymptotes where the function shoots to ±∞.

Discontinuities can be identified by checking if the limit equals the function value.

Example: f(x) = (x² − 1)/(x − 1) has a removable discontinuity at x = 1 (hole), while f(x) = 1/(x − 1) has an infinite discontinuity (asymptote).

Continuity at a point

A function is continuous at x = a if all three conditions hold:

  1. f(a) is defined
  2. lim(x→a) f(x) exists
  3. lim(x→a) f(x) = f(a)

Intuitively, you can draw the graph without lifting your pencil at that point.

Example: f(x) = x² is continuous everywhere because for any a, the limit as x→a equals a².

Continuity over an interval

A function is continuous on an interval if it is continuous at every point in that interval. For closed intervals [a,b], we also check one-sided continuity at the endpoints. Polynomial, exponential, and trigonometric functions are continuous wherever they are defined.

Example: f(x) = √x is continuous on [0, ∞) but not on (−∞, ∞) because it is not defined for x < 0.

Removing discontinuities

Removable discontinuities (holes) can be "fixed" by redefining the function value at that point. Find the limit value and set f(a) equal to that limit. This only works for removable discontinuities, not jumps or infinite discontinuities.

Example: f(x) = (x² − 1)/(x − 1) has a hole at x = 1. Define f(1) = 2 (the limit), and the function becomes continuous.

Infinite limits

An infinite limit occurs when a function grows unboundedly. Notation: lim(x→a) f(x) = ∞ means f(x) grows arbitrarily large as x approaches a. Similarly, lim(x→a) f(x) = −∞ means f(x) becomes arbitrarily negative. These often occur at vertical asymptotes where a denominator equals zero.

Example: lim(x→0⁺) 1/x = ∞ and lim(x→0⁻) 1/x = −∞.

Limits at infinity

These describe function behavior as x approaches ±∞. Notation: lim(x→∞) f(x) = L means f(x) approaches L as x grows large. To evaluate, compare the degrees of numerator and denominator in rational functions: if degrees are equal, the limit is the ratio of leading coefficients; if the numerator degree is larger, the limit is ±∞.

Example: lim(x→∞) (3x² + 1)/(2x² − 5) = 3/2 because the highest-degree terms dominate.

Intermediate value theorem

If f is continuous on [a,b] and N is any value between f(a) and f(b), then there exists at least one c in (a,b) such that f(c) = N. In plain language: a continuous function passes through every output value between its starting and ending values. This is useful for proving solutions exist without finding them explicitly.

Example: f(x) = x² is continuous on [1,3]. Since f(1) = 1 and f(3) = 9, there must be some x where f(x) = 5.

Unit 2: Derivatives, definition and basic rules

Derivative definition

The derivative f'(a) measures the instantaneous rate of change of f at x = a. Formally, f'(a) = lim(h→0) [f(a+h) − f(a)]/h. Geometrically, it is the slope of the tangent line at (a, f(a)). The derivative tells you how fast the output is changing at that instant. If the limit does not exist, the function is not differentiable at that point.

Example: For f(x) = x², f'(2) = lim(h→0) [(2+h)² − 4]/h = lim(h→0) [4h + h²]/h = 4.

Estimating derivatives

You can estimate the derivative at a point by computing the slope between two nearby points on the curve, using the secant line slope (f(a+h) − f(a))/h for small h. The symmetric difference quotient [f(a+h) − f(a−h)]/(2h) often gives a better approximation. Graphically, draw the tangent line and measure its slope.

Example: To estimate f'(1) for f(x) = x³, use f(1.1) ≈ 1.331 and f(0.9) ≈ 0.729, then [1.331 − 0.729]/0.2 ≈ 3.

Differentiability

A function is differentiable at a point if the derivative exists there. Continuous functions are not always differentiable—cusps, corners, and vertical tangents create points where the derivative does not exist. If a function has a break (discontinuity), it is not differentiable there. Conversely, if f is differentiable at a, then f is continuous at a.

Example: f(x) = |x| is continuous at x = 0 but not differentiable there because the left and right derivatives differ.

Power rule

If f(x) = xⁿ, then f'(x) = n·xⁿ⁻¹. This is one of the most useful rules in calculus. The power rule works for any real exponent, including negative and fractional powers.

Example: f(x) = x⁵ has f'(x) = 5x⁴. f(x) = x^(−2) has f'(x) = −2x^(−3) = −2/x³.

Product rule

If f(x) = u(x)·v(x), then f'(x) = u'(x)·v(x) + u(x)·v'(x). Memorize this as "first times derivative of second plus second times derivative of first." This rule handles products of two functions and extends to products of more functions.

Example: f(x) = x²·sin(x). Using product rule: f'(x) = 2x·sin(x) + x²·cos(x).

Quotient rule

If f(x) = u(x)/v(x), then f'(x) = [u'(x)·v(x) − u(x)·v'(x)] / [v(x)]². A common mnemonic is "low d-high minus high d-low, square the bottom and away we go." Always check whether the numerator can be simplified or rewritten to use product rule instead.

Example: f(x) = x/(x² + 1). Using quotient rule: f'(x) = [(1)(x² + 1) − (x)(2x)] / (x² + 1)² = (1 − x²) / (x² + 1)².

Derivative rules (constant, sum, difference, constant multiple)

  • Constant: If f(x) = c, then f'(x) = 0
  • Sum/Difference: (f ± g)' = f' ± g'
  • Constant Multiple: (k·f)' = k·f'

These rules let you differentiate term by term and factor out constants for simplification.

Example: f(x) = 3x² + 2x + 5 gives f'(x) = 6x + 2 + 0 = 6x + 2.

Derivatives of cos(x), sin(x), e^x, and ln(x)

Key derivatives:

  • d/dx[sin(x)] = cos(x)
  • d/dx[cos(x)] = −sin(x)
  • d/dx[e^x] = e^x
  • d/dx[ln(x)] = 1/x

The exponential function e^x is notable because it is its own derivative, making it solutions to many important differential equations.

Example: f(x) = sin(x) + e^x gives f'(x) = cos(x) + e^x.

Derivatives of tan(x), cot(x), sec(x), and csc(x)

  • d/dx[tan(x)] = sec²(x)
  • d/dx[cot(x)] = −csc²(x)
  • d/dx[sec(x)] = sec(x)tan(x)
  • d/dx[csc(x)] = −csc(x)cot(x)

These follow from the quotient rule applied to tan = sin/cos, etc. Notice the pattern: the derivatives of the "co-" functions have a negative sign.

Example: f(x) = tan(x) gives f'(x) = sec²(x).

Unit 3: Chain rule

Chain rule

If f(x) = g(h(x)) (a composition), then f'(x) = g'(h(x))·h'(x). In words, differentiate the outer function (keeping the inner function), multiply by the derivative of the inner function. This is one of the most important rules because many real functions are compositions. Think of it as "peel off one layer, multiply by the derivative of that layer."

Example: f(x) = sin(x²). Outer: sin(u) → cos(u), Inner: u = x² → du/dx = 2x. So f'(x) = cos(x²)·2x = 2x·cos(x²).
Practice:
1) f(x) = (3x + 1)⁵, then f'(x) has outer exponent factor

Implicit differentiation

When a relation is given implicitly (like x² + y² = 25) rather than as y = f(x), differentiate both sides with respect to x, treating y as a function of x. When you differentiate y terms, include dy/dx using the chain rule. Solve for dy/dx algebraically. This technique finds slopes without explicitly solving for y.

Example: x² + y² = 25. Differentiating: 2x + 2y(dy/dx) = 0, so dy/dx = −x/y.

Differentiating inverse functions

If y = f(x) and x = g(y) are inverses, then g'(y) = 1/f'(x). In general, if f is differentiable and invertible, then [f⁻¹]'(y) = 1/f'(f⁻¹(y)). This relationship comes from the chain rule applied to f(f⁻¹(x)) = x.

Example: f(x) = 2x + 3 has inverse f⁻¹(x) = (x − 3)/2. Since f'(x) = 2, we have [f⁻¹]'(x) = 1/2.

Derivatives of inverse trigonometric functions

Key derivatives:

  • d/dx[arcsin(x)] = 1/√(1 − x²)
  • d/dx[arccos(x)] = −1/√(1 − x²)
  • d/dx[arctan(x)] = 1/(1 + x²)

These follow from implicit differentiation or the inverse function derivative formula. When combined with the chain rule, these allow you to differentiate more complex expressions.

Example: f(x) = arctan(3x). Using chain rule: f'(x) = 1/(1 + (3x)²) · 3 = 3/(1 + 9x²).

Second derivatives

The second derivative f''(x) is the derivative of the first derivative, measuring how the rate of change itself is changing. Notation: f''(x), d²y/dx², or d/dx[dy/dx]. The second derivative relates to concavity: if f''(x) > 0, the curve is concave up (U-shaped); if f''(x) < 0, it is concave down (∩-shaped). Second derivatives appear in acceleration (if position is f(x)).

Example: f(x) = x³ gives f'(x) = 3x² and f''(x) = 6x.

Unit 4: Applications of derivatives

Meaning of the derivative in context

The derivative f'(a) is an instantaneous rate of change. In context, this might be velocity (rate of position change), marginal cost (rate of cost change per item), population growth rate, or any other rate. Always connect the derivative to the real-world interpretation: what is changing, with respect to what, and what does the sign mean?

Example: If h(t) is height at time t, then h'(t) is the rate of height change, measured in units per second.

Straight line motion

For motion along a line with position s(t):

  • Velocity: v(t) = s'(t)
  • Acceleration: a(t) = v'(t) = s''(t)
  • Speed: |v(t)| (always positive)

Interpretation:

  • Object moves forward when v > 0
  • Object moves backward when v < 0
  • Object is stationary when v = 0

Acceleration direction and magnitude determine whether motion is speeding up or slowing down.

Example: s(t) = −16t² + 20t (height in feet). Then v(t) = −32t + 20 ft/s, and v = 0 at t = 5/8 seconds (peak height).

Non-motion applications of derivatives

Derivatives apply wherever rates matter: revenue and marginal revenue, cost and marginal cost, concentration of pollutants, temperature change, etc. Set up the relevant equation, differentiate, and interpret the result in the context. Common problems ask for maximum profit or minimum cost.

Example: If profit P(x) = −x² + 100x, find the production level x that maximizes profit by setting P'(x) = 0.

Related rates

Related rates problems involve quantities that change over time and are related by an equation.

Method:

  1. Identify the constraint equation relating all variables
  2. Differentiate implicitly with respect to time t
  3. Substitute known values
  4. Solve for the desired rate (usually dr/dt)
Example: A ladder 10 ft long leans against a wall. If the bottom slides away at 2 ft/s, how fast is the top sliding down when the base is 6 ft from the wall? Use x² + y² = 100, differentiate to get 2x(dx/dt) + 2y(dy/dt) = 0, then solve for dy/dt.

L'Hôpital's rule: Applications of derivatives

When lim(x→a) f(x)/g(x) is indeterminate (0/0 or ∞/∞), you can compute:

lim(x→a) f'(x)/g'(x)

instead, provided this limit exists. This replaces an algebraically difficult limit with derivatives.

Note: Some indeterminate forms like 0·∞ can be rewritten as 0/0 or ∞/∞ first.

Example: lim(x→0) sin(x)/x gives 0/0, so apply L'Hôpital's: lim(x→0) cos(x)/1 = 1.

L'Hôpital's rule: composite exponential functions

Forms like 1^∞, 0^0, or ∞^0 are indeterminate. To handle these, use logarithms: if y = f(x)^g(x), then ln(y) = g(x)·ln(f(x)), which may become a 0·∞ form, then 0/0 or ∞/∞ suitable for L'Hôpital's rule. Differentiate, apply L'Hôpital's if needed, then exponentiate to recover y.

Example: lim(x→∞) (1 + 1/x)^x. Take ln: lim(x→∞) x·ln(1 + 1/x) = lim(x→∞) ln(1 + 1/x)/(1/x), which is 0/0. Apply L'Hôpital's to find the limit is ln(e) = 1, so the original limit is e.

Unit 5: Analyzing functions

Mean value theorem

If f is continuous on [a,b] and differentiable on (a,b), then there exists at least one c in (a,b) such that f'(c) = [f(b) − f(a)]/(b − a). Geometrically, the instantaneous rate of change at some point equals the average rate of change over the interval. This theorem guarantees that level roads must have a point where you match the average slope.

Example: f(x) = x² on [1,3]. Average rate = (9 − 1)/(3 − 1) = 4. At c = 2, f'(2) = 4.

Extreme value theorem and critical points

The extreme value theorem states that a continuous function on a closed interval [a,b] attains its maximum and minimum. Extrema occur at:

  • Critical points: where f'(x) = 0 or is undefined
  • Endpoints: a and b

To find extrema: compute critical points, evaluate f at each critical point and endpoint, then compare values.

Example: f(x) = x² − 4x on [0,3]. f'(x) = 2x − 4 = 0 at x = 2. Check f(0) = 0, f(2) = −4, f(3) = −3. Minimum is −4 at x = 2.

Decreasing and increasing intervals

  • Increasing: If f'(x) > 0 on an interval, f(x) is increasing there
  • Decreasing: If f'(x) < 0 on an interval, f(x) is decreasing there

Process: Find critical points, place them on a number line, test the sign of f' in each region, and mark accordingly. This analysis helps sketch the graph and understand the function's behavior.

Example: f(x) = x³ − 3x has f'(x) = 3x² − 3 = 3(x² − 1). Critical points: x = ±1. f' > 0 when x < −1 or x > 1 (increasing); f' < 0 when −1 < x < 1 (decreasing).

Relative (local) extrema

A local maximum is where f(c) ≥ f(x) for all x in some neighborhood of c.

A local minimum is where f(c) ≤ f(x) nearby.

Local extrema occur at critical points. Use the first derivative test:

  • If f' changes from positive to negative: local max
  • If f' changes from negative to positive: local min
Example: f(x) = x³ − 3x has a local max at x = −1 (where f' goes from + to −) and local min at x = 1.

Absolute (global) extrema

Global extrema are the largest and smallest values of f over an entire domain. On a closed interval, they occur at critical points or endpoints. On an open interval or the entire real line, analyze the behavior as x → ±∞. Global extrema may not exist if the function is unbounded.

Example: f(x) = x² on ℝ has a global minimum at x = 0 (value 0) and no global maximum (grows to ∞).

Concavity and inflection points intro

Concavity describes the curving direction of a graph:

  • Concave up: f''(x) > 0 (U-shape)
  • Concave down: f''(x) < 0 (∩-shape)

An inflection point is where concavity changes. It occurs where f''(x) = 0 or is undefined, and the sign of f'' changes. At inflection points, the curve transitions from bending one way to bending the other.

Example: f(x) = x³ has f''(x) = 6x. For x < 0, f'' < 0 (concave down); for x > 0, f'' > 0 (concave up). Inflection point at x = 0.

Analyzing concavity and inflection points

Compute f''(x), find points where f''(x) = 0 or is undefined, place these on a number line, test the sign of f'' in each region, and mark each region as concave up or down. Inflection points are where the sign changes. Sketch the graph showing these concavity regions for a more accurate picture.

Example: f(x) = x⁴ − 4x² has f''(x) = 12x² − 8. Solve 12x² − 8 = 0 to get x = ±√(2/3). Analyze concavity in three regions.

Second derivative test

At a critical point c where f'(c) = 0, examine f''(c):

  • If f''(c) > 0: local minimum
  • If f''(c) < 0: local maximum
  • If f''(c) = 0: test is inconclusive (use first derivative test instead)

The second derivative test is faster than the first test when f'' is easy to compute.

Example: f(x) = x³ − 3x has f'(x) = 3x² − 3, critical points at x = ±1. f''(x) = 6x, so f''(−1) = −6 < 0 (local max) and f''(1) = 6 > 0 (local min).

Unit 6: Parametric equations, polar coordinates, and vector valued functions

Intro to parametric equations

Parametric equations describe a curve using a parameter t:

  • x = f(t)
  • y = g(t)

As t varies, the point (x, y) traces out the curve. Parametric form is useful for:

  • Motion problems: where t represents time
  • Curves that fail vertical line test: like circles or loops

To find dy/dx, use the chain rule: dy/dx = (dy/dt) / (dx/dt). Parametric curves are oriented—they show the direction of motion as t increases.

Example: x = cos(t), y = sin(t) for t ∈ [0, 2π] traces a unit circle counterclockwise as t increases.

Second derivatives of parametric equations

For parametric curves, the second derivative d²y/dx² measures concavity. Use d²y/dx² = d/dx(dy/dx) = d/dt(dy/dx) ÷ (dx/dt). This requires computing the derivative of (dy/dt)/(dx/dt) with respect to t, then dividing by dx/dt. It is tedious but necessary for analyzing concavity on parametric curves.

Example: For x = t², y = t³, compute dy/dx = (3t²)/(2t) = 3t/2, then d²y/dx² involves further differentiation and division.

Vector-valued functions

A vector-valued function assigns a vector to each input:

  • 2D: r(t) = ⟨f(t), g(t)⟩
  • 3D: r(t) = ⟨f(t), g(t), h(t)⟩

Key concepts:

  • Derivative: r'(t) = ⟨f'(t), g'(t)⟩ (tangent vector)
  • Tangent vector shows direction and speed of motion
  • Used for curves and central to multivariable calculus
Example: r(t) = ⟨2cos(t), 3sin(t)⟩ traces an ellipse. Then r'(t) = ⟨−2sin(t), 3cos(t)⟩.

Planar motion

Planar motion is motion in 2D described by r(t) = ⟨x(t), y(t)⟩.

Key quantities:

  • Velocity: v(t) = r'(t) = ⟨x'(t), y'(t)⟩ (points in direction of motion)
  • Speed: |v(t)| = √[x'(t)² + y'(t)²]
  • Acceleration: a(t) = v'(t) = r''(t) = ⟨x''(t), y''(t)⟩

Acceleration has components: tangential (speeding up/down) and normal (curving).

Example: Projectile motion: r(t) = ⟨v₀cos(θ)·t, v₀sin(θ)·t − (1/2)g·t²⟩. Velocity and acceleration describe the trajectory.

Polar functions

Polar coordinates: (r, θ) describe points as:

  • r: distance from the origin
  • θ: angle from the positive x-axis

A polar function r = f(θ) describes a curve where r depends on θ. To find dy/dx in polar form:

  1. Convert to parametric: x = r cos(θ), y = r sin(θ)
  2. Apply parametric derivative formula: dy/dx = (dy/dθ) / (dx/dθ)

Polar curves can have loops, spirals, and other shapes not easily describable in Cartesian form.

Example: r = 1 + cos(θ) is a cardioid. As θ goes from 0 to 2π, this traces a heart-shaped curve. At θ = 0, r = 2; at θ = π, r = 0.
