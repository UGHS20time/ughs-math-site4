Unit 1: Polynomial arithmetic

Intro to polynomials arithmetic

A polynomial is an expression made up of terms added or subtracted, each term being a constant times a variable raised to a non-negative integer power. The highest exponent determines the degree of the polynomial.

Example: 3x^2 + 2x - 5 is a polynomial of degree 2.

Average rate of change of polynomials

The average rate of change of a polynomial f(x) from x=a to x=b is [f(b) - f(a)] / (b - a). This is the slope of the secant line over the interval.

Example: f(x) = x^2, from x=1 to x=3: [9-1]/2 = 4.

Unit 2: Complex numbers

Imaginary Unit i

The imaginary unit is a special number denoted i, defined by the property i² = -1. It is the solution to equations like x² = -1, which have no real solutions. The imaginary unit allows us to work with square roots of negative numbers.

Powers of i cycle every four steps:

  • i¹ = i
  • i² = -1
  • i³ = i² · i = -1 · i = -i
  • i⁴ = (i²)² = (-1)² = 1
  • i⁵ = i (cycle repeats)

Key insight: To find i^n for large n, divide n by 4 and evaluate i^(remainder).

Example: i² = -1; i³ = -i; 2i · 3i = 6i² = 6(-1) = -6. Find i^17: 17 ÷ 4 = 4 remainder 1, so i^17 = i^1 = i.

Intro to complex numbers

A complex number has the form a + bi, where a and b are real numbers. The number a is called the real part, and b is called the imaginary part. Complex numbers extend the number system to include square roots of negative numbers.

  • Real part: The coefficient a (the part without i)
  • Imaginary part: The coefficient b (the number multiplying i)
  • Purely real: When b = 0, e.g., 5 + 0i = 5
  • Purely imaginary: When a = 0, e.g., 0 + 3i = 3i

Note: The set of complex numbers includes all real numbers (as the special case where the imaginary part is 0). Real numbers are a subset of complex numbers.

Example: 3 + 2i (a = 3, b = 2); -1 - 4i (a = -1, b = -4); 7i (a = 0, b = 7); 5 (a = 5, b = 0).

Complex plane

The complex plane (also called the Argand diagram) is a 2D coordinate system for visualizing complex numbers. The horizontal axis represents the real part, and the vertical axis represents the imaginary part. Any complex number a + bi is plotted as the point (a, b).

Key features:

  • Real axis: Horizontal axis (y = 0) containing purely real numbers
  • Imaginary axis: Vertical axis (x = 0) containing purely imaginary numbers
  • Distance from origin: The modulus (or absolute value) of a + bi is |a + bi| = √(a² + b²)
Example: 2 + 3i is plotted at point (2, 3). Its distance from origin is √(2² + 3²) = √13 ≈ 3.61. The point -1 - 2i is at (-1, -2).

Adding and subtracting complex numbers

To add or subtract complex numbers, combine real parts with real parts and imaginary parts with imaginary parts separately. Think of (a + bi) and (c + di) as separate real and imaginary components.

Addition: (a + bi) + (c + di) = (a + c) + (b + d)i

Subtraction: (a + bi) - (c + di) = (a - c) + (b - d)i

Example: (3 + 2i) + (1 - 4i) = (3 + 1) + (2 - 4)i = 4 - 2i. Another example: (5 + 3i) - (2 + 7i) = (5 - 2) + (3 - 7)i = 3 - 4i.

Multiplying complex numbers

Multiply complex numbers using the distributive property (FOIL), then replace i² with -1. The key is remembering that i² = -1, which converts imaginary terms to real numbers.

Process:

  1. Apply FOIL: (a + bi)(c + di)
  2. Multiply: ac + adi + bci + bdi²
  3. Replace i² with -1: ac + adi + bci - bd
  4. Combine real and imaginary parts: (ac - bd) + (ad + bc)i
Example: (2 + i)(1 - 3i):
  • = 2(1) + 2(-3i) + i(1) + i(-3i)
  • = 2 - 6i + i - 3i²
  • = 2 - 6i + i - 3(-1)
  • = 2 - 5i + 3 = 5 - 5i

Quadratic equations with complex solutions

A quadratic equation ax² + bx + c = 0 has complex solutions when the discriminant (b² - 4ac) is negative. When this happens, the quadratic formula produces square roots of negative numbers, leading to complex solutions. These complex solutions always come in conjugate pairs.

Conjugate pairs: If p + qi is a solution, then p - qi is also a solution. They differ only in the sign of the imaginary part.

Using the quadratic formula with negative discriminant:

  1. Calculate discriminant b² - 4ac (which is negative)
  2. The square root √(negative number) = i√(positive number)
  3. Apply quadratic formula: x = [-b ± i√(|discriminant|)] / (2a)
  4. Solutions come as a pair: p + qi and p - qi
Example: x² + 4 = 0: x² = -4, so x = ±√(-4) = ±2i. The solutions are 0 + 2i and 0 - 2i (conjugate pair). Another example: x² - 2x + 5 = 0 has discriminant 4 - 20 = -16. Solutions: x = [2 ± √(-16)] / 2 = [2 ± 4i] / 2 = 1 ± 2i.

Unit 3: Polynomial Factorization

Factoring monomials

Factoring a monomial means writing it as a product of its factors. Break coefficients into primes and use exponent rules.

Example: 12x^3 = 2*2*3*x*x*x.

Greatest common factor

The GCF is the largest factor shared by all terms. It includes the greatest shared coefficient and lowest powers of shared variables.

Example: GCF of 6x^2 and 9x = 3x.

Taking common factors

Factor out the GCF from each term. This simplifies expressions and reveals structure for further factoring.

Example: 4x^2 + 6x = 2x(2x + 3).

Factoring higher degree polynomials

Look for patterns (difference of squares, cubes, grouping) to factor polynomials of degree 3 or higher. Synthetic division can also help find factors.

Example: x^3 - 8 = (x - 2)(x^2 + 2x + 4).

Factoring using structure

Sometimes, substitute a variable to reveal a pattern, then factor and substitute back. This works well for expressions like x^4 + 5x^2 + 4.

Example: x^4 - 16 = (x^2 - 4)(x^2 + 4) = (x - 2)(x + 2)(x^2 + 4).

Polynomial identities

Identities like (a + b)^2 = a^2 + 2ab + b^2 help factor and expand polynomials. Memorizing these saves time.

Example: (x + 3)^2 = x^2 + 6x + 9.

Geometric series

A geometric series is the sum of a geometric sequence: a + ar + ar^2 + ... + ar^{n-1}. The sum formula depends on the ratio r.

Example: 2 + 4 + 8 + 16 = 2(1 - 2^4)/(1 - 2) = 30.

Unit 4: Polynomial Division

Dividing polynomials by x

Divide each term by x. For example, (2x^3 + 4x^2) / x = 2x^2 + 4x. Make sure x is not zero.

Dividing quadratics by linear factors

Use long division or synthetic division to divide quadratics by linear factors. This helps rewrite rational expressions or find factors.

Example: (x^2 + 3x + 2) / (x + 1) = x + 2.

Dividing polynomials by linear factors

Similar to above, use long or synthetic division for higher degree polynomials. Always check your quotient by multiplying back.

Remainder Theorem

The remainder when f(x) is divided by (x - a) is f(a). This makes evaluating remainders fast without full division.

Example: f(x) = x^2 + 2x + 3, divided by x - 1: remainder is f(1) = 1 + 2 + 3 = 6.

Unit 5: Polynomial Graphs

Zeros of polynomials

Zeros are x-values where the polynomial equals zero (roots). They are the x-intercepts of the graph.

Example: x^2 - 4 = 0 → x = 2, -2.

Positive and negative intervals of polynomials

Intervals where the polynomial is above (positive) or below (negative) the x-axis. Sign changes occur at zeros and turning points.

Example: For f(x) = x^2 - 4, f(x) > 0 for x < -2 or x > 2.

Unit 6: Rational exponents and radicals

Rational exponents

Rational exponents are fractional powers: a^{m/n} = n-th root of a^m. They connect radicals and exponential form.

Example: 8^{1/3} = 2.

Properties of rational exponents

Same rules as integer exponents: a^{m} * a^{n} = a^{m+n}, (a^{m})^{n} = a^{mn}. Apply these before simplifying radicals.

Evaluating exponents & radicals

Convert between radical and exponent form to evaluate. This makes it easier to use exponent rules.

Example: √9 = 9^{1/2} = 3.

Equivalent forms of exponential expressions

Write expressions in different but equal forms: √x = x^{1/2}. This helps compare and simplify expressions.

Solving exponential equations using properties of exponents

Rewrite both sides with the same base, then set exponents equal. If you cannot match bases, use logarithms instead.

Example: 2^{x} = 8 → 2^{x} = 2^{3} → x = 3.

Unit 7: Exponential models

Interpreting the rate of change

In exponential models, the rate of change is multiplicative, not additive. Each step scales the value by a constant factor.

Example: y = 2^x doubles for each increase of 1 in x.

Building exponential models according to rate of change

Use y = a*b^x, where b is the growth/decay factor. The initial value a is the value when x = 0.

Example: A population triples each year: y = 100*3^x.

Advanced interpretation of exponential models

Analyze how changes in the base or exponent affect the model's growth or decay. Small changes in the base can lead to large long-term differences.

Unit 8: Logarithms

Introduction to logarithms

Logarithms are the inverse operation of exponentials. They answer the question: "What exponent must I raise the base to in order to get this number?" Logarithms and exponentials are inverse functions, just as square roots and squaring are inverse.

Definition: logb(a) = c means bc = a (where b > 0, b ≠ 1, a > 0)

Key interpretations:

  • logb(a) is the exponent you need
  • b is the base
  • a is the argument (the number inside the logarithm)
  • c is the result (the exponent)

Common logarithmic bases:

  • log10(x) = common logarithm (base 10, denoted log x)
  • log2(x) = binary logarithm (base 2, used in computer science)
  • ln(x) = natural logarithm (base e ≈ 2.718, denoted loge) Most important in calculus
Example: log2(8) = 3 because 23 = 8. log10(1000) = 3 because 103 = 1000. ln(e) = 1 because e1 = e.

Constant e and natural logarithm

The number e (approximately 2.71828) is one of the most important constants in mathematics. It arises naturally in continuous growth and decay problems. The natural logarithm ln(x) is the logarithm with base e: ln(x) = loge(x).

Why e is special:

  • e is irrational and transcendental (not a root of any polynomial with integer coefficients)
  • The function ex has the unique property that its derivative equals itself: d/dx(ex) = ex
  • Continuous compound interest uses e
  • Exponential growth/decay models naturally use e

Natural logarithm properties:

  • ln(e) = 1 (because e1 = e)
  • ln(1) = 0 (because e0 = 1)
  • ln(ex) = x (they are inverse functions)
  • eln(x) = x
Example: ln(7.389) ≈ 2 because e² ≈ 7.389. A population growing continuously at 5% per year follows P(t) = P0e0.05t.

Properties of logarithms

Logarithms have algebraic properties that make them powerful for simplifying and solving exponential equations. These properties mirror the rules of exponents because logarithms are exponents.

Logarithm properties (base b > 0, b ≠ 1):

  • Product rule: logb(xy) = logb(x) + logb(y)
  • Quotient rule: logb(x/y) = logb(x) - logb(y)
  • Power rule: logb(xk) = k · logb(x)
  • Identity: logb(b) = 1 and logb(1) = 0
  • Reciprocal: logb(1/x) = -logb(x)

Using properties to expand: Write a single logarithm as a sum/difference of simpler logs. Example: log(xy) = log(x) + log(y)

Using properties to condense: Combine multiple logarithms into a single log. Example: log(x) + log(y) = log(xy)

Example of expanding: log2(8x3) = log2(8) + log2(x3) = 3 + 3log2(x). Example of condensing: ln(x) + ln(5) - ln(2) = ln(5x/2).

Change of base formula for logarithms

The change of base formula allows you to convert a logarithm from one base to another. This is essential because most calculators only have log (base 10) and ln (base e).

Change of base formula:

logb(a) = logc(a) / logc(b)

where b, c > 0 and b, c ≠ 1. You can choose any base c that your calculator supports (typically base 10 or base e).

Most common forms:

  • logb(a) = ln(a) / ln(b) (using natural logarithm)
  • logb(a) = log(a) / log(b) (using common logarithm base 10)

Why this works: If logb(a) = x, then bx = a. Taking logc of both sides: x · logc(b) = logc(a), so x = logc(a) / logc(b).

Example: Find log2(10) using a calculator:
  • log2(10) = ln(10) / ln(2) ≈ 2.3026 / 0.6931 ≈ 3.322
  • Check: 23.322 ≈ 10 ✓

Solving exponential equations with logarithms

When exponential equations have different bases or non-integer exponents, logarithms are essential for solving. The key strategy is to take the logarithm of both sides, which allows you to bring the exponent down as a coefficient using the power rule.

Steps to solve exponential equations:

  1. Isolate the exponential term if possible
  2. Take the logarithm (any base, but usually ln or log) of both sides
  3. Use the power rule: log(ax) = x · log(a)
  4. Solve for the variable algebraically
  5. Check your answer (optional but recommended)

Strategy when bases match: If 2x = 8, write both sides as powers of 2: 2x = 23, so x = 3. This avoids logarithms.

Strategy when bases differ: Use logarithms. Example: 3x = 10 requires logs.

Example: Solve 2x = 10:
  • ln(2x) = ln(10)
  • x · ln(2) = ln(10)
  • x = ln(10) / ln(2) ≈ 3.322
  • Check: 23.322 ≈ 10 ✓
Another example: Solve e2x = 20:
  • ln(e2x) = ln(20)
  • 2x = ln(20)
  • x = ln(20) / 2 ≈ 1.497

Unit 9: Transformation of functions

Shifting functions

f(x) + k shifts up/down; f(x + h) shifts left/right. Positive k moves the graph up, positive h moves it left.

Example: y = x^2 + 3 shifts y = x^2 up by 3.

Reflecting functions

-f(x) reflects over x-axis; f(-x) reflects over y-axis. Reflections change the orientation but not the shape.

Example: y = -x^2 is a reflection of y = x^2 over the x-axis.

Symmetry of functions

Even functions: f(-x) = f(x). Odd: f(-x) = -f(x). Even functions are symmetric about the y-axis, odd about the origin.

Example: y = x^2 is even; y = x^3 is odd.

Scaling functions

af(x) stretches vertically; f(bx) compresses horizontally. Values of a and b control how steep or wide the graph becomes.

Example: y = 2x^2 is a vertical stretch of y = x^2.

Graphs of square and cubre root functions

y = √x is a half-parabola; y = ∛x is an S-shaped curve. The square root starts at (0,0) and only takes nonnegative inputs.

Graphs of exponential functions

y = a^x grows rapidly for a > 1, decays for 0 < a < 1. Exponential graphs cross the y-axis at 1.

Graphs of logarithmic funcitons

y = log_b(x) increases slowly, undefined for x ≤ 0. The graph passes through (1,0).

Unit 10: Equations

Rational equations

Equations with fractions whose numerators and/or denominators are polynomials. Always check for excluded values that make denominators zero.

Example: 1/(x+1) = 2/(x-1).

Square root equations

Equations involving square roots. Isolate the root, then square both sides. Check solutions to avoid extraneous answers.

Example: √(x+3) = 5 → x+3 = 25 → x = 22.

Extraneous solutions

Squaring both sides can introduce false solutions. Always check answers in the original equation. This is especially common with radicals and rational equations.

Cube-root equations

Equations with cube roots. Cube both sides to solve. Unlike square roots, cubing does not introduce sign ambiguity.

Example: ∛(x-1) = 3 → x-1 = 27 → x = 28.

Quadratic systems

Systems with at least one quadratic equation. Solve by substitution or graphing. Solutions are intersection points of the graphs.

Solving equations by graphing

Graph both sides and find intersection points. Each intersection gives a solution that satisfies both equations.

Unit 11: Trigonometry

Unit circle introduction

The unit circle is a circle of radius 1 centered at the origin. It helps define sine and cosine for all angles. Each point on the circle corresponds to an angle.

Example: The point (√2/2, √2/2) on the unit circle corresponds to 45° (π/4 radians).

Radians

Radians are another way to measure angles: 180° = π radians. Many trig formulas are simplest in radians.

Example: 90° = π/2 radians.

Pythagorean Identity

sin^2(θ) + cos^2(θ) = 1 for any angle θ. This comes from the unit circle equation x^2 + y^2 = 1.

Trigonometric value for special angles

Special angles: 0°, 30°, 45°, 60°, 90° (and their radian equivalents) have known sine and cosine values. Memorizing these helps solve many problems quickly.

Example: sin(30°) = 1/2, cos(60°) = 1/2.

Amplitude, midline, period

For y = a sin(bx) + d, amplitude = |a|, period = 2π/|b|, midline = y = d. These features describe the size, speed, and vertical shift of the wave.
