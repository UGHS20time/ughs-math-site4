Unit 1: Composite and inverse functions

Composing functions

Function composition means using the output of one function as the input to another. Written as (f ∘ g)(x) = f(g(x)), composition chains functions together in a specific order. The key insight is that the order matters—f ∘ g is generally different from g ∘ f.

How to compute (f ∘ g)(x) = f(g(x)):

  1. Evaluate the inside function first: compute g(x)
  2. Take that result and use it as input to f: compute f(result)
  3. Simplify the overall expression if possible

Domain of composition: x must be in the domain of g, AND g(x) must be in the domain of f.

Example: If f(x) = 2x + 1 and g(x) = x², then:
  • f(g(x)) = f(x²) = 2(x²) + 1 = 2x² + 1
  • g(f(x)) = g(2x + 1) = (2x + 1)² = 4x² + 4x + 1
  • Notice f ∘ g ≠ g ∘ f (composition order matters!)

Invertible functions

A function has an inverse if and only if it is one-to-one (injective)—meaning each output value comes from exactly one input value. Visually, this means the function passes the horizontal line test: every horizontal line crosses the graph at most once.

Why invertibility matters: Inverse functions "undo" the original function, swapping the roles of input and output.

Testing for invertibility:

  • Graph the function and apply the horizontal line test
  • If any horizontal line crosses the graph more than once, the function is NOT one-to-one
  • For many non-invertible functions, you can restrict the domain to make them invertible (e.g., f(x) = x² is not one-to-one on all reals, but is one-to-one on [0, ∞))
Example: f(x) = 3x - 2 is invertible because it's a non-horizontal line (strictly increasing). Any horizontal line crosses it exactly once. f(x) = x² is NOT invertible on all reals because horizontal lines (like y = 4) cross it twice. But it IS invertible on [0, ∞).
Practice:
1) Is f(x) = x² invertible over all real numbers? (answer: no)

Inverse functions

The inverse function f⁻¹ reverses the action of f: if f takes x to y, then f⁻¹ takes y back to x. The graphs of f and f⁻¹ are reflections of each other across the line y = x. The domain of f⁻¹ is the range of f, and vice versa.

Finding the inverse algebraically:

  1. Start with y = f(x)
  2. Swap x and y to get x = f(y)
  3. Solve for y in terms of x
  4. This gives y = f⁻¹(x)

Important notation: f⁻¹(x) means "the inverse function of f applied to x", NOT "1/f(x)". These are completely different!

Example: If f(x) = 2x - 5, find f⁻¹(x):
  • Start: y = 2x - 5
  • Swap: x = 2y - 5
  • Solve: x + 5 = 2y, so y = (x + 5)/2
  • Therefore: f⁻¹(x) = (x + 5)/2

Verifying inverse functions by composition

To confirm that two functions f and g are inverses of each other, you must verify both composition equations on their appropriate domains:

  • f(g(x)) = x for all x in the domain of g
  • g(f(x)) = x for all x in the domain of f

If both are true, then f and g are inverses. If either fails, they are not inverses. This composition property directly follows from the definition—each function undoes the other when applied in sequence.

Why we need both directions: Checking only one direction is insufficient. For example, f(g(x)) = x might hold without g(f(x)) = x also holding, meaning they're not true inverses.

Example: Verify that f(x) = 2x + 1 and g(x) = (x - 1)/2 are inverses:
  • f(g(x)) = f((x-1)/2) = 2((x-1)/2) + 1 = (x-1) + 1 = x ✓
  • g(f(x)) = g(2x+1) = ((2x+1)-1)/2 = 2x/2 = x ✓
  • Both check out, so they are inverses.
Practice:
1) If f(x) = x - 3 and g(x) = x + 3, is f(g(x)) = x? (answer: yes)

Unit 2: Complex numbers

Complex plane

The complex plane uses the horizontal axis for the real part and the vertical axis for the imaginary part. A + bi is plotted as the point (a, b). This lets you visualize complex numbers like points or vectors.

Example: 3 - 2i is plotted at (3, -2).
Practice:
1) Plot 1 + 4i as (1, 4). Is the imaginary part 4? (answer: yes)

Distance and midpoint of complex numbers

Distance between complex numbers is the distance between their points in the plane. Midpoint is the average of real and imaginary parts. Use the distance formula just like with ordered pairs.

Example: Between 1 + i and 5 + 3i, midpoint is (3, 2) or 3 + 2i.

Example: (x^2 - 4)/(x^2 - x) = (x+2)(x-2)/(x(x-1)).

End behavior of rational functions

End behavior depends on the degrees of numerator and denominator. Compare leading terms to find horizontal or slant asymptotes. If the numerator degree is larger by one, expect a slant asymptote.

Example: (2x^2 + 1)/(x^2 - 3) has end behavior near y = 2.

Unit 4: Conic sections

Intro to conic sections

Conic sections are curves formed by slicing a cone: circles, ellipses, parabolas, and hyperbolas. Each has a standard equation and key features. You identify a conic by the signs and placement of x^2 and y^2 terms.

Example: x^2 + y^2 = 9 is a circle of radius 3.

Center and radii of an ellipse

An ellipse centered at (h,k) has equation (x-h)^2/a^2 + (y-k)^2/b^2 = 1. The semi-axes are a and b. The larger denominator tells the direction of the major axis.

Example: (x-1)^2/9 + (y+2)^2/4 = 1 has center (1,-2), radii 3 and 2.

Foci of an ellipse

For an ellipse, c^2 = a^2 - b^2. The foci are at (h ± c, k) or (h, k ± c) depending on orientation. The foci lie on the major axis inside the ellipse.

Example: For a=5, b=3, c = 4.

Intro to hyperbolas

A hyperbola has equation (x-h)^2/a^2 - (y-k)^2/b^2 = 1 or the reverse. It opens left-right or up-down. The signs tell which variable opens in the positive and negative direction.

Example: x^2/4 - y^2/9 = 1 opens left-right.

Foci of a hyperbola

For a hyperbola, c^2 = a^2 + b^2. The foci are farther from the center than the vertices. This is why the branches approach but never touch the asymptotes.

Example: For a=3, b=4, c = 5.

Hyperbolas not centered at the origin

Shifted hyperbolas use (x-h) and (y-k) in the equation. The center moves to (h,k). Asymptotes shift the same way as the center.

Example: (x-2)^2/9 - (y+1)^2/4 = 1 has center (2,-1).

Unit 5: Vectors

Intro to vectors

Vectors have magnitude and direction and are often written as bold letters or arrows. In the plane, a vector can be written as . Vectors can represent movement, force, or displacement.

Example: v = <3, -2> represents 3 units right and 2 units down.

Vector components

Components are the horizontal and vertical parts of a vector. They show how far the vector moves in each direction. You can treat components like ordered pairs for calculations.

Example: v = <4, 1> has components 4 and 1.

Magnitude of vectors

Magnitude is the length of the vector: |v| = sqrt(x^2 + y^2). It is always nonnegative and represents distance from the origin.

Example: |<3,4>| = 5.

Scalar multiplication

Multiply each component by the scalar. This changes magnitude but not direction (unless the scalar is negative). A negative scalar flips the direction by 180 degrees.

Example: 2<3, -1> = <6, -2>.

Vector addition and subtraction

Add or subtract vectors by combining components. This gives the resultant vector. Graphically, this matches the tip-to-tail method.

Example: <2,3> + <1,-4> = <3,-1>.

Direction of vectors

The direction angle is found with tan(theta) = y/x, measured from the positive x-axis. Adjust the angle based on the quadrant of the vector.

Example: For <3,3>, theta = 45 degrees.

Adding vectors in magnitude and direction form

Example: Add r=5 at 0 deg and r=5 at 90 deg to get <5,5>.

Unit 6: Matrices

Intro to matrices

A matrix is a rectangular array of numbers with rows and columns. Matrices represent data and linear transformations. The size is written as rows by columns.

Example: [[1, 2], [3, 4]] is a 2x2 matrix.

Using matrices to represent data

Matrices store information like tables. Rows and columns can represent categories or variables. This makes it easy to apply operations to entire datasets.

Example: A 2x3 matrix can represent two students and three test scores.

Multiplying matrices by scalars

Multiply every entry in the matrix by the scalar. The matrix size does not change.

Example: 2[[1,2],[3,4]] = [[2,4],[6,8]].

Adding and subtracting matrices

Only matrices of the same size can be added or subtracted. Add entries in the same position. The result has the same dimensions.

Example: [[1,2],[3,4]] + [[2,1],[0,1]] = [[3,3],[3,5]].

Properties of matrix addition and scalar multiplication

Matrix addition is commutative and associative, and scalar multiplication distributes over addition. These properties let you simplify expressions just like with numbers.

Example: A + B = B + A, and k(A + B) = kA + kB.

Matrix inverses

A matrix inverse A^-1 satisfies A A^-1 = I. Only square matrices with nonzero determinant are invertible. Inverses undo the effect of the matrix transformation.

Example: The inverse of [[1,0],[0,1]] is itself.

Solving linear systems with matrices

Systems can be written as AX = B. If A is invertible, then X = A^-1 B. This method is most useful for larger systems and consistent data.

Example: Use A^-1 to solve two equations in two variables.

Unit 7: Probability and combinatorics

Venn diagrams and the addition rule

The addition rule for probability is P(A or B) = P(A) + P(B) - P(A and B). Venn diagrams help visualize overlap. Subtract the overlap so it is not counted twice.

Example: If P(A)=0.6, P(B)=0.5, P(A and B)=0.2, then P(A or B)=0.9.

Multiplication rule for probabilities

P(A and B) = P(A) * P(B|A). If events are independent, P(B|A) = P(B). Independence means one event does not change the probability of the other.

Example: Rolling two dice: P(both 6) = 1/6 * 1/6 = 1/36.

Permutations

Permutations count ordered arrangements. For n items taken r at a time: nPr = n!/(n-r)!. Order matters, so swapping two items gives a different outcome.

Example: 5P2 = 5*4 = 20.

Combinations

Combinations count unordered selections: nCr = n!/(r!(n-r)!). Order does not matter, so the same group is counted once.

Example: 5C2 = 10.

Probability using combinatorics

Use combinations or permutations to count favorable outcomes over total outcomes. Choose permutations when order matters and combinations when it does not.

Example: Choosing 2 hearts from 5 cards: favorable = 13C2, total = 52C2.

Unit 8: Series

Geometric series

Sum of a geometric series: S_n = a(1 - r^n)/(1 - r) for r != 1. Here a is the first term and r is the common ratio.

Example: 2 + 4 + 8 + 16 = 30.

The binomial theorem

(a + b)^n expands into a sum with binomial coefficients. These coefficients come from Pascal's triangle. Each term has decreasing powers of a and increasing powers of b.

Example: (x + 1)^3 = x^3 + 3x^2 + 3x + 1.

Arithmetic series

Sum of an arithmetic series: S_n = n/2 (first + last). This works because pairs of terms have the same sum.

Example: 1 + 2 + 3 + 4 + 5 = 15.

Unit 9: Limits and continuity

Defining limits using limit notation

Limit notation describes the value a function approaches as x approaches a number. The symbol is lim x->a f(x). The function value at x = a can be different from the limit.

Example: lim x->2 (x^2) = 4.

Estimating values from graphs

Use a graph to see what y-value the curve approaches near a point, even if the function is not defined there. The left and right sides should approach the same value for the limit to exist.

Example: If the graph approaches y=3 as x->1, the limit is 3.

Estimating limit from tables and graphs

Tables show values close to the target x-value from both sides. Use them with graphs to estimate limits. Compare left-hand and right-hand values to check consistency.

Example: If f(1.9)=3.8 and f(2.1)=4.2, the limit near 2 is about 4.

Limit properties

Limits distribute over addition, subtraction, multiplication, and division (when the denominator is not zero). These properties let you break a hard limit into easier ones.

Example: lim (f+g) = lim f + lim g.

Algebraic manipulation

Factor, simplify, or rationalize to evaluate limits that give 0/0 forms. After simplifying, substitute again to finish.

Example: lim x->2 (x^2-4)/(x-2) = lim x->2 (x+2) = 4.

Removing discontinuities

Removable discontinuities occur when a factor cancels. Define the function at that point to make it continuous. The limit value tells what to fill in.

Example: f(x) = (x^2-1)/(x-1) has a hole at x=1 but can be defined as 2 there.
