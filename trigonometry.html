Unit 1: Right triangles

Ratios in right triangles

In a right triangle, the sides are related by special ratios that depend on the angles. These ratios are scale-independent — meaning similar triangles (triangles with the same angles but different sizes) have identical ratio values. This fundamental property allows us to use trigonometry to solve any right triangle.

The three parts of a right triangle relative to an angle θ:

  • Opposite: The side across from angle θ
  • Adjacent: The side next to angle θ (but not the hypotenuse)
  • Hypotenuse: The longest side, opposite the right angle
Example: In a right triangle with angle θ = 30°, if the opposite side is 3 and hypotenuse is 5, then sin(30°) = 3/5 = 0.6. In a similar triangle scaled up where opposite is 6 and hypotenuse is 10, sin(30°) is still 6/10 = 0.6.

Intro to trigonometric ratios

The three main trigonometric ratios are sine (sin), cosine (cos), and tangent (tan). Each ratio compares two sides of a right triangle relative to a specific angle θ. Memory aid: SOH-CAH-TOA.

  • sin(θ) = opposite/hypotenuse (Sine = Opposite/Hypotenuse)
  • cos(θ) = adjacent/hypotenuse (Cosine = Adjacent/Hypotenuse)
  • tan(θ) = opposite/adjacent (Tangent = Opposite/Adjacent)

Important: These ratios are properties of the angle, not the triangle. Any two right triangles with the same angle θ will have the same sine, cosine, and tangent values.

Example: In a 3-4-5 right triangle, if θ is the angle opposite the side of length 3: sin(θ) = 3/5, cos(θ) = 4/5, tan(θ) = 3/4.

Solving for a side in a right triangle using the trigonometric ratios

To find an unknown side in a right triangle, identify which ratio (sin, cos, or tan) connects your known angle, known side, and unknown side. Then use algebra to solve.

Steps:

  1. Label opposite, adjacent, and hypotenuse for the given angle
  2. Choose the correct ratio (SOH-CAH-TOA) that uses both the known and unknown sides
  3. Write an equation with the ratio
  4. Solve for the unknown side
Example: In a right triangle, angle A = 30°, hypotenuse = 10, find the opposite side.
  • sin(30°) = opposite/10
  • opposite = 10 × sin(30°) = 10 × 0.5 = 5
Another example: If cos(30°) = x/10 and cos(30°) = √3/2, then x = 10 × √3/2 = 5√3 ≈ 8.66.

Solving for an angle in a right triangle using trigonometric ratios

To find an unknown angle when you know two sides, use the inverse trigonometric functions (arcsin, arccos, arctan or written as sin⁻¹, cos⁻¹, tan⁻¹). These "reverse" the trigonometric ratios to give you the angle.

Process:

  1. Identify which two sides you know relative to the unknown angle θ
  2. Form the appropriate ratio (sin, cos, or tan)
  3. Apply the inverse function to solve for θ
  4. Result is typically in degrees or radians (check your calculator)
Example: If opposite = 5 and hypotenuse = 10, find θ:
  • sin(θ) = 5/10 = 0.5
  • θ = sin⁻¹(0.5) = 30°
Another example: If adjacent = 4 and opposite = 3, find θ:
  • tan(θ) = 3/4 = 0.75
  • θ = tan⁻¹(0.75) ≈ 36.87°

Sine and cosine of complementary angles

In a right triangle, the two non-right angles are complementary (they sum to 90°). This creates a special relationship between sine and cosine called cofunction identities.

  • sin(90° − θ) = cos(θ)
  • cos(90° − θ) = sin(θ)

Why this works: If you rotate perspective to the complementary angle, opposite becomes adjacent and vice versa.

Example:
  • sin(30°) = cos(60°) = 0.5 (since 30° + 60° = 90°)
  • sin(40°) = cos(50°)
  • cos(25°) = sin(65°)

Reciprocal ratios

Three additional trigonometric ratios are the reciprocals of the main three. They are less commonly used but appear in identities and advanced problems.

  • csc(θ) = 1/sin(θ) = hypotenuse/opposite (cosecant)
  • sec(θ) = 1/cos(θ) = hypotenuse/adjacent (secant)
  • cot(θ) = 1/tan(θ) = adjacent/opposite (cotangent)

Key insight: You don't need to memorize these separately—they're just inverses of the main ratios. If sin(θ) = 5/13, then csc(θ) = 13/5.

Example: In a 3-4-5 right triangle where sin(θ) = 3/5:
  • csc(θ) = 1/sin(θ) = 1/(3/5) = 5/3
  • cos(θ) = 4/5, so sec(θ) = 5/4
  • tan(θ) = 3/4, so cot(θ) = 4/3

Unit 2: Trigonometric functions

Intro to unit circle

The unit circle is a circle with radius 1 centered at the origin of a coordinate plane. It is the fundamental tool for extending trigonometry beyond right triangles to all angles (including negative and reflex angles). Every point on the unit circle can be written as (cosθ, sinθ), where θ is the angle measured counterclockwise from the positive x-axis.

Key insight: For any angle θ:

  • The x-coordinate of the point is cos(θ)
  • The y-coordinate of the point is sin(θ)
  • The point is always distance 1 from the origin (by definition of unit circle)

Special points on unit circle:

  • 0° (or 0 rad): (1, 0) → cos(0°) = 1, sin(0°) = 0
  • 90° (or π/2 rad): (0, 1) → cos(90°) = 0, sin(90°) = 1
  • 180° (or π rad): (-1, 0) → cos(180°) = -1, sin(180°) = 0
  • 270° (or 3π/2 rad): (0, -1) → cos(270°) = 0, sin(270°) = -1
Example: At θ = 45°, the point is (√2/2, √2/2), so cos(45°) = √2/2 and sin(45°) = √2/2.

Radians

Radians are an angle measurement unit based on arc length rather than degrees. One radian is the angle subtended by an arc equal in length to the radius. The relationship between radians and degrees is:

  • π radians = 180°
  • To convert degrees to radians: multiply by π/180°
  • To convert radians to degrees: multiply by 180°/π

Common angle conversions:

  • 45° = π/4 radians
  • 60° = π/3 radians
  • 90° = π/2 radians
  • 360° = 2π radians

Why radians matter: Radians are the standard in calculus and advanced mathematics because arc length, angular velocity, and many formulas simplify beautifully when using radians.

Example: Convert 30° to radians: 30° × (π/180°) = π/6 radians. Convert π/4 to degrees: (π/4) × (180°/π) = 45°.

Pythagorean Identity

The Pythagorean Identity is one of the most important trigonometric identities:

sin²(θ) + cos²(θ) = 1

This identity holds for all angles θ. It comes directly from the equation of the unit circle: x² + y² = 1, where x = cos(θ) and y = sin(θ).

Using the Pythagorean Identity:

  • If you know sin(θ), you can find cos(θ): cos²(θ) = 1 - sin²(θ)
  • If you know cos(θ), you can find sin(θ): sin²(θ) = 1 - cos²(θ)
  • It helps simplify trigonometric expressions
  • It provides verification of trigonometric values
Example: If cos(θ) = 3/5, find sin(θ):
  • sin²(θ) + (3/5)² = 1
  • sin²(θ) = 1 - 9/25 = 16/25
  • sin(θ) = ±4/5

Special trigonometric values in first quadrant

Certain angles have exact trigonometric values that deserve memorization because they appear frequently. These values come from two special right triangles: the 45-45-90 and 30-60-90 triangles.

Table of special angles (degrees and radians):

Angle sin(θ) cos(θ) tan(θ)
0° (0) 0 1 0
30° (π/6) 1/2 √3/2 √3/3
45° (π/4) √2/2 √2/2 1
60° (π/3) √3/2 1/2 √3
90° (π/2) 1 0 undefined

Derivation hint: The 45-45-90 triangle has sides 1-1-√2 (all sides divided by √2). The 30-60-90 triangle has sides 1-√3-2. These give the exact values above.

Example: sin(45°) = √2/2 (not 0.707, which is an approximation). cos(60°) = 1/2 exactly.

Trigonometric values in unit circle

Every point (x, y) on the unit circle represents an angle θ, and from that point you can read off the trigonometric values. The beauty of the unit circle is that it extends trigonometric values to all angles (positive, negative, and beyond 360°), and it shows how signs change in different quadrants.

For any angle θ on the unit circle (point (x, y)):

  • cos(θ) = x (first coordinate)
  • sin(θ) = y (second coordinate)
  • tan(θ) = sin(θ)/cos(θ) = y/x (if x ≠ 0)

Signs by quadrant:

  • Quadrant I (0° to 90°): sin > 0, cos > 0, tan > 0
  • Quadrant II (90° to 180°): sin > 0, cos < 0, tan < 0
  • Quadrant III (180° to 270°): sin < 0, cos < 0, tan > 0
  • Quadrant IV (270° to 360°): sin < 0, cos > 0, tan < 0

Memory aid (ASTC): "All Students Take Calculus" reminds you which functions are positive in each quadrant: All (Q1), Sine (Q2), Tangent (Q3), Cosine (Q4).

Example: At θ = 150° (in Quadrant II): sin(150°) = 1/2 (positive), cos(150°) = -√3/2 (negative), tan(150°) = -1/√3 (negative).

Amplitude, midline and period

Trigonometric functions create periodic waves with predictable patterns. For a function of the form y = a sin(b(x - c)) + d (or cosine), each coefficient controls a different aspect of the wave.

  • Amplitude = |a|: Measures the maximum distance from the midline to the peak. The wave oscillates between d - |a| and d + |a|.
  • Period = 2π/|b|: The horizontal distance for one complete cycle. For regular sine/cosine, period = 2π.
  • Midline = y = d: The vertical shift. The wave oscillates around this horizontal line.
  • Phase shift = c: The horizontal shift. Positive c shifts right, negative shifts left.

Example interpretations:

  • y = 3 sin(x): Amplitude 3, period 2π, midline y = 0
  • y = sin(2x): Amplitude 1, period π (twice as fast), midline y = 0
  • y = sin(x) + 5: Amplitude 1, period 2π, midline y = 5 (shifted up)
  • y = 2 sin(3x - π) + 1: Amplitude 2, period 2π/3, phase shift π/3 right, midline y = 1
Example: For y = 4 sin(2(x - π/4)) + 1:
  • Amplitude = 4 (wave goes from -3 to 5)
  • Period = 2π/2 = π (one cycle every π units)
  • Midline = 1 (centered at y = 1)
  • Shifted π/4 units to the right

Unit 3: Non-right triangles

Law of sines

The Law of Sines relates the sides and angles of any triangle (not just right triangles). It states that the ratio of each side to the sine of its opposite angle is constant:

sin(A)/a = sin(B)/b = sin(C)/c

Where capital letters denote angles and lowercase letters denote the sides opposite those angles.

When to use Law of Sines:

  • ASA (Angle-Side-Angle): Two angles and one side known
  • AAS (Angle-Angle-Side): Two angles and a non-included side known
  • SSA (Angle-Angle-Side, ambiguous case): Two sides and a non-included angle known (may have 0, 1, or 2 solutions)

Steps to solve:

  1. Set up the ratio equation
  2. Cross multiply if necessary
  3. Solve for the unknown
  4. Check for ambiguous cases (SSA can have multiple solutions)
Example: Triangle with A = 30°, B = 45°, a = 5. Find b:
  • sin(30°)/5 = sin(45°)/b
  • 0.5/5 = (√2/2)/b
  • b = 5 × sin(45°)/sin(30°) = 5 × (√2/2)/0.5 = 5√2 ≈ 7.07

Law of cosines

The Law of Cosines is a generalization of the Pythagorean Theorem that works for any triangle. It relates all three sides and one angle:

c² = a² + b² − 2ab cos(C)

Where C is the angle between sides a and b, and c is the opposite side.

When to use Law of Cosines:

  • SAS (Side-Angle-Side): Two sides and the included angle known
  • SSS (Side-Side-Side): All three sides known (need to find an angle)

Finding angles with Law of Cosines: Rearrange to cos(C) = (a² + b² - c²)/(2ab), then use inverse cosine.

Connection to Pythagorean Theorem: If C = 90°, then cos(90°) = 0, and we get c² = a² + b² (the Pythagorean Theorem).

Example: Triangle with a = 3, b = 4, C = 60°. Find c:
  • c² = 3² + 4² - 2(3)(4)cos(60°)
  • c² = 9 + 16 - 24(0.5) = 25 - 12 = 13
  • c = √13 ≈ 3.61

Solving general triangles

Solving a general (non-right) triangle means finding all unknown sides and angles given some starting information. The strategy depends on what you know (ASA, AAS, SAS, SSS, or SSA).

Decision tree for triangle solving:

  • ASA or AAS (two angles given): Use Law of Sines directly
  • SAS (two sides + included angle): Use Law of Cosines to find 3rd side, then Law of Sines for remaining angles
  • SSS (all three sides): Use Law of Cosines rearranged to find each angle
  • SSA (two sides + non-included angle, ambiguous case): Use Law of Sines; check if 0, 1, or 2 solutions exist

Ambiguous case (SSA): Given sides a, b and angle A opposite side a. If a < b sin(A), no triangle exists. If a = b sin(A), one right triangle. If b sin(A) < a < b, two triangles. If a ≥ b, one triangle.

Steps for any case:

  1. Identify what type of case you have
  2. Choose appropriate law (Sines or Cosines)
  3. Solve for one unknown
  4. Use remaining information to find remaining unknowns
  5. Verify your answer using angle sum = 180°

Unit 4: Trigonometric equations and identities

Inverse trigonometric functions

The inverse trigonometric functions (also called arcsine, arccosine, arctangent) reverse the operation of trigonometric functions. They find the angle when you know the trigonometric ratio.

  • sin⁻¹(x) or arcsin(x): Returns the angle whose sine is x. Range: [-90°, 90°] or [-π/2, π/2]
  • cos⁻¹(x) or arccos(x): Returns the angle whose cosine is x. Range: [0°, 180°] or [0, π]
  • tan⁻¹(x) or arctan(x): Returns the angle whose tangent is x. Range: (-90°, 90°) or (-π/2, π/2)

Important: Each inverse function returns a principal value (one specific angle). If other angles with the same trigonometric value are needed, you must find them using reference angles and quadrant information.

Example finding all solutions: If sin(θ) = 0.5, then sin⁻¹(0.5) = 30° (principal value). But sin(θ) = 0.5 also when θ = 150°, 390°, etc.

Example: If cos(θ) = 0.5, then θ = cos⁻¹(0.5) = 60° (principal). Other solutions: ..., -300°, -60°, 60°, 300°, ...

Angle addition identities

The angle addition formulas allow you to find the trigonometric values of sums (or differences) of angles, even if the individual angles aren’t special angles. These formulas are crucial for expanding, simplifying, and solving trigonometric equations.

Sine addition formulas:

  • sin(A + B) = sin(A)cos(B) + cos(A)sin(B)
  • sin(A − B) = sin(A)cos(B) − cos(A)sin(B)

Cosine addition formulas:

  • cos(A + B) = cos(A)cos(B) − sin(A)sin(B)
  • cos(A − B) = cos(A)cos(B) + sin(A)sin(B)

Tangent addition formula:

  • tan(A + B) = [tan(A) + tan(B)] / [1 − tan(A)tan(B)]
Example: Find sin(75°) = sin(45° + 30°):
  • sin(75°) = sin(45°)cos(30°) + cos(45°)sin(30°)
  • = (√2/2)(√3/2) + (√2/2)(1/2)
  • = √6/4 + √2/4 = (√6 + √2)/4 ≈ 0.966

Using trigonometric identities

Trigonometric identities are equations that are true for all values of the variable. They are powerful tools for simplifying expressions, solving equations, and proving other identities.

Common trigonometric identities:

  • Pythagorean: sin²(θ) + cos²(θ) = 1
  • Reciprocal: csc(θ) = 1/sin(θ), sec(θ) = 1/cos(θ), cot(θ) = 1/tan(θ)
  • Quotient: tan(θ) = sin(θ)/cos(θ), cot(θ) = cos(θ)/sin(θ)
  • Double angle: sin(2θ) = 2sin(θ)cos(θ); cos(2θ) = cos²(θ) - sin²(θ)
  • Cofunction: sin(90° - θ) = cos(θ), cos(90° - θ) = sin(θ)

Strategy for simplifying expressions and solving equations:

  1. Look for patterns matching known identities
  2. Rewrite everything in terms of sin and cos if possible
  3. Factor and combine like terms
  4. Apply identities to reduce complexity
  5. Check for extraneous solutions when solving
